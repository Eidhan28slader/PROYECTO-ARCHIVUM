Normas de uso y ética de Archivum

Reglas de uso de la plataforma

Archivum es una plataforma inspirada en Pinterest donde los usuarios pueden compartir contenido multimedia, ideas e imágenes. La idea es que sea un espacio agradable, dinámico y útil para todos, así que es importante usarla con respeto y responsabilidad.

Política de contenido

En esta plataforma no se permite contenido ofensivo, discriminatorio, violento o que pueda afectar a otros usuarios. Tampoco se permite publicar cosas que invadan la privacidad de otras personas o que generen un ambiente incómodo dentro de la comunidad.

Además, los comentarios también deben mantenerse dentro de un tono respetuoso. Si alguien intenta publicar contenido que no cumple con estas reglas, el sistema puede bloquearlo automáticamente.

Responsabilidad del usuario

Cada usuario es responsable de lo que publica, comenta o comparte dentro de Archivum. Por eso, se espera que cada persona use la plataforma de forma consciente y respetando a los demás. Si un usuario encuentra contenido inapropiado, puede denunciarlo para que sea revisado.

Privacidad y protección de datos

Para realizar ciertas acciones dentro de la plataforma, como publicar o comentar, se necesita tener una cuenta iniciada. Esto ayuda a llevar un mejor control de las acciones de cada usuario y también aporta más seguridad dentro del sistema.

Por otro lado, los archivos multimedia se almacenan en Amazon S3, y el acceso a esos recursos se maneja con permisos definidos para proteger la información y evitar accesos no autorizados.

Convivencia dentro de la plataforma

Más allá de que el sistema tenga validaciones y restricciones, la idea principal es fomentar un espacio donde los usuarios puedan compartir contenido de forma sana, creativa y respetuosa. No se trata solo de cumplir reglas, sino de cuidar la experiencia de todos los que usan la plataforma.
